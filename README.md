# Solaire
A python script editor for DCC development.

A PySide6 code editor for scripting in DCCs with file management and intellisense.
Still has some work to be done.

## Roadmap

- [x] Minimap
- [x] Std Lib and Project Code Intellisense
- [x] Git Push + Pull
- [x] Output Terminal
- [x] Customizable Syntax Highlighting
- [x] JSON Syntax Highlighting
- [x] YAML Syntax Highlighting
- [x] Basic Theme Support
- [x] Customizable Shortcuts
- [ ] Powershell
- [ ] DCC Built-in Intellisense (like `maya.cmds` or `unreal`)
- [ ] Git Branch Management
- [ ] Project Management
- [ ] Plugin Support
- [ ] Plugin API
- [ ] Detached Process Support
